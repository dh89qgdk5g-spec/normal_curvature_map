## TO INSTALL:
1. Download main.py
2. Open Blender
3. Go to: Edit -> Preferences -> Add-ons -> Install
4. Select the Add-on file from its download location
5. Click on install Add-on

## INSTRUCTIONS:
1. On object Mode select your desired object
2. Cpen the side bar/N-panel by pressing "n"
3. set your contrast and scale parameters of the heatmap with the sliders
4. click on calculate curvatures

### NOTE: 
a. when changing visualization parameters you must re-calculate curvatures 

b. the weight paint infotmation can be accesed by selecting "Curvature" vertex group on the data section of the properties pannel
